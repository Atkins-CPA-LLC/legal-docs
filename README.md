# Legal Docs

Shared legal documents for Atkins CPA applications.

| File | Purpose |
| --- | --- |

## Current Application Coverage

The current documents cover an invitation-only AI Financial Analyst application that syncs QuickBooks Online financial data, stores it in Supabase, and uses third-party AI providers through OpenRouter to answer user questions about authorized financial data.

## Maintenance Notes

- Keep document titles, application names, contact details, and last-updated dates aligned across files.
- Update this repository when application behavior, data storage, AI provider usage, authentication, retention, or user rights change.
- These files are documentation artifacts only; legal review is still required before relying on them for production use.
